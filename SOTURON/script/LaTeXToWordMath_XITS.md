# LaTeXToWordMath_XITS

**概要**
このマクロは Word 文書内のインライン LaTeX スタイル（`$...$`）を数式オブジェクトに変換し、文書内のすべての数式フォントを **XITS Math** に統一します。

**前提**
- システムに **XITS Math** フォントがインストールされていること。

**導入手順**
1. Word を開き、`Alt + F11` で VBA エディタを起動します。
2. メニューの **ファイル > ファイルのインポート** から `LaTeXToWordMath_XITS.bas` を選択してインポートします。
3. `Alt + F8` でマクロ一覧を開き、`LaTeXToWordMath_XITS` を実行します。

**注意事項**
- 実行前にドキュメントのバックアップを取ることを推奨します。
- 現在はインライン数式（`$...$`）のみに対応しています。ディスプレイ数式（`$$...$$`）等が必要な場合は拡張可能です。
