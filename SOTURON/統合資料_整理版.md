# 第一部：プリント基板材料の基礎知識

## 1. プリント基板の基本構成と機能

### 1.1 プリント基板の種類

プリント基板は、電子機器の基本構成部材です。構造により以下に分類されます：

**① 片面基板**
- 基板の片面のみに導体回路を形成
- コスト低減が必要な大量生産品に使用

**② 両面基板**
- 基材の両面に導体回路を形成
- 高密度配線が可能で、産業用機器や高信頼性機器で使用

**③ 多層基板**
- 基材内部と表層に複数（3層以上）の導体回路を形成
- 軽薄短小、高密度実装が要求される用途に使用

### 1.2 基板材料の選択基準

プリント基板内での信号伝搬速度は、基板材料の比誘電率により決定されます：

$$v = K \cdot \frac{c}{\sqrt{\varepsilon_r'}}$$

ここで：
- $v$：伝搬速度
- $c$：光速
- $\varepsilon_r'$：比誘電率
- $K$：定数

**結論**：誘電率を低くすれば伝搬速度が上がります。5G（30GHz程度）などの高周波信号伝送では、低誘電率材料が求められます。

## 3. 誘電体の物理的特性

### 3.1 誘電体の定義と電気的特性

**誘電体とは**：外部電界が印加されたときにエネルギーを蓄積できる材料

**電気的特性の分類**：

```
誘電体の電気的特性
├─ 誘電性
│  ├─ 誘電分極（誘電率 εr'）
│  └─ 誘電損失（εr''）→ 誘電正接 tanδ
└─ 電気絶縁性
   ├─ 電気伝導
   └─ 絶縁破壊
```

### 3.2 誘電分極のメカニズム

物質が電界に置かれると、内部の電荷分布が変化します。これを誘電分極といい、3つの種類があります：

**① 電子分極**
- 原子核を中心とした電子雲がわずかに変位
- 最も高速（紫外線領域まで応答）

**② イオン分極**
- イオン結晶内の正・負イオンが相互に変位
- 赤外線領域程度の周波数まで応答

**③ 双極子分極（配向分極）**
- 永久双極子を持つ分子が電界方向に配向
- マイクロ波領域程度の周波数まで応答

### 3.3 複素誘電率の定義

交流電界が印加されると、分極の変化は電界の変化に対して位相遅れを伴います。

電界：$E = E_0 \exp(j\omega t)$

電束密度：$D = D_0 \exp[j(\omega t - \delta)]$

複素誘電率：
$$\dot{\varepsilon} = \varepsilon_r' - j\varepsilon_r'' = \frac{D_0}{E_0}\cos\delta - j\frac{D_0}{E_0}\sin\delta$$

**実部** $\varepsilon_r'$：比誘電率（材料の分極のしやすさを表す）

**虚部** $\varepsilon_r''$：誘電損失（熱エネルギーへの変換を表す）

### 3.4 誘電損失と誘電正接

**誘電正接**（loss tangent）：
$$\tan\delta = \frac{\varepsilon_r''}{\varepsilon_r'}$$

**意味**：
- 誘電損失の程度を示す指標
- tanδが小さいほど損失の少ない材料
- 高周波回路設計では最重要パラメータ

## 4. FR-4基板の異方性

### 4.1 構造の特徴

FR-4基板は、ガラス繊維の層を複数重ねてエポキシ樹脂で固めた構造になっています：

```
FR-4基板の層構造
┌─────────────────────┐
│  銅箔（表面）      │
├─────────────────────┤
│ ガラス繊維層       │
│  + エポキシ樹脂     │
│ (複数層重ねた構造) │
├─────────────────────┤
│ ガラス繊維層       │
│  + エポキシ樹脂     │
├─────────────────────┤
│  銅箔（裏面）      │
└─────────────────────┘
```

### 4.2 異方性の発生メカニズム

この層状構造により、複素誘電率が方向に依存します：

- **垂直方向**（厚さ方向）：ガラス繊維とエポキシ樹脂の配合比に大きく依存
- **水平方向**（表面平行）：異なる特性を示す

**測定上の重要性**：
- 垂直方向の測定：平衡円板共振器法（BCDR）
- 水平方向の測定：フリースペース法など

# 第二部：複素誘電率の測定技術

## 1. 測定方法の分類と特徴

複素誘電率の測定方法は、測定対象の周波数帯と材料特性により使い分けられます。

### 1.1 代表的な3つの測定法

```
                             誘電正接 tanδ
                             10^(-1)～10^(-5)
                                ↑
                          ┌─────┼─────┐
        100 MHz           │     │     │         100 GHz+
        ────────────────────────────────────────────→ 周波数
                          │     │     │

    集中定数法         共振法       フリースペース法
    (低周波・簡便)    (中周波・高精度)  (高周波・高損失)

    ├─ SOLT校正        ├─ BCDR法      ├─ ホーンアンテナ
    ├─ ~1GHz          ├─ ~100GHz    ├─ ~170GHz+
    └─ 大きな試料     └─ 小さな試料   └─ 広い試料

```

### 1.2 各測定法の詳細

#### **① 集中定数法**
- **原理**：測定試料を2枚の電極で挟んでコンデンサを形成
- **測定器**：インピーダンスアナライザ
- **周波数範囲**：100 Hz～1 GHz
- **メリット**：簡便、精度高い
- **デメリット**：低周波帯のみ

**計測式**：
$$\varepsilon_r' = \frac{tC}{A\varepsilon_0}, \quad \varepsilon_r'' = \frac{tG}{\omega A \varepsilon_0}$$

#### **② 反射伝送法**
- **原理**：試料に入射した電磁波の反射・伝送特性（Sパラメータ）から誘電率を導出
- **測定器**：ベクトルネットワークアナライザ（VNA）
- **周波数範囲**：GHz～100 GHz（広帯域）
- **メリット**：高損失材料に適する、広い周波数範囲
- **デメリット**：低損失材料での精度劣化

#### **③ 共振法（平衡円板共振器法：BCDR）**
- **原理**：共振器内の特定電磁界モード（TM0m0モード）の共振周波数変化から誘電率を決定
- **周波数範囲**：1 GHz～170 GHz（複数モードの活用で広帯域化）
- **メリット**：低損失材料で高精度、複数周波数同時測定
- **デメリット**：共振器の製造精度が重要

---

## 2. 平衡円板共振器法（BCDR）の詳細

### 2.1 BCDR法の原理

#### **構成要素**
```
      ┌─────────────────┐
      │  上部銅板       │
      ├─────────────────┤
      │  測定試料(2枚)  │
      │  中央:銅箔円板  │
      ├─────────────────┤
      │  下部銅板       │
      └─────────────────┘
          ↓同軸励振↓
      中央の励振孔から
      TM0m0モードを励振
```

#### **励振方式**
- 同軸線路で中央励振孔を励振
- 0.8mm～1.0mm径の超微細同軸線路を使用
- TM0m0モード（中央に電界集中）を選択的に励振

#### **測定可能なモード**
- 基本モード：TM010
- 高次モード：TM020, TM030, ... , TM0m0
- 単一の共振器で複数周波数での測定が可能

### 2.2 TM0m0モードの電磁界分布

**TM010モードの場合**：

```
     ┌─────────────┐
     │  磁界(円周) │  ← φ方向成分
     │  ┌─────┐   │
     │  │ ●E │   │  ← 電界（中央に集中）
     │  └─────┘   │
     └─────────────┘
     
    ●：電界（z方向）
    ○：磁界（φ方向、円周状に回転）
```

### 2.3 複素誘電率の算出方法

**① 比誘電率εr'の決定**

特性方程式：
$$\det[d, r, M, \varepsilon_r, f_0] = 0$$

ここで：
- $d$：円板銅箔の厚さ
- $r$：共振器の半径
- $M$：励振空洞の寸法
- $\varepsilon_r$：比誘電率（未知数）
- $f_0$：測定された共振周波数

**② 誘電正接（tanδ）の決定**

$$\tan\delta = \left( \frac{1}{Q_u} - \frac{1}{Q_c} \right) \left( 1 + \frac{W_2^e}{W_1^e} \right)$$

ここで：
- $Q_u$：無負荷時のQ値
- $Q_c$：負荷時のQ値
- $W_1^e, W_2^e$：電磁エネルギー

### 2.4 BCDR法の利点と制限

**利点**：
- 複数モードで多周波数測定可能
- 低損失材料で高精度
- 試料が小さくてよい

**制限**：
- 導電率が周波数に比例して増加すると、60GHz以上で精度劣化
- 高い製造精度が必要

## 4. BCDR法とフリースペース法の比較

### 4.1 測定精度の周波数特性

```
測定精度
  ↑
100│     ┌─────────────BCDR法─────────────┐
   │     │ （低損失材料で高精度）        │
精 │     │                             ↘
度 │     │                              ╲
   │     │                               ╲
   │  ┌──┴───────────────────────────────┘
   │  │   フリースペース法
   │  │  （高周波で精度向上）
 0 └──┴─────────────────────────────────→ 周波数
   0      60GHz        100GHz      170GHz+
```

### 4.2 各測定法の適用周波数域

**北原研究結果（FR-4基板）**：
- BCDR法：～60 GHz（精度確保可能）
- フリースペース法：60 GHz～

**大日方研究結果**：
- BCDR法：～60 GHz
- フリースペース法：26.5～90 GHz（60GHz以上で安定）

**結論**：2つの方法を組み合わせることで26GHz～170GHz以上をカバー可能

# 第三部：測定結果と考察

## 1. 北原研究：複素誘電率と伝送特性

### 1.1 BCDR法による複素誘電率測定結果

#### **比誘電率εr'の周波数特性**
- 周波数増加とともに低下傾向（誘電分散）
- メーカ間での差異：最大0.1程度
- 同一メーカでも厚さにより約0.05の差

#### **誘電損失εr''の特性**
- メーカ間で不規則に変動
- 測定値のばらつき：最大0.03程度
- tanδはほぼ0.02前後に収まる（公称値に近い）

### 1.2 差動線路の伝送特性測定

#### **測定対象**
- マイクロストリップ構造の差動線路
- 複数のFR-4基板（厚さ0.3～0.5mm）
- 周波数：0～25 GHz

#### **測定結果（Sdd21）**
```
  -12 dB
   ─┼────────────────────→ Frequency 25GHz
    │ ╲ L6504C2（減衰大）
    │  ╲
 -6 dB   ╲═════════════════ 他の基板
    │     ╲               （減衰小）
    │      ╲
   0 dB    └────────────
    0         周波数
```

- 全基板で高周波ほど信号減衰が大きい
- L6504C2だけ他より大きな減衰を示す
- 基板間の減衰差：最大2 dB程度（減衰倍率1.2倍）

### 1.3 伝送特性（Sdd21）の算出と比較

#### **減衰の構成要素**
$$\alpha = \alpha_c + \alpha_d$$

- $\alpha_c$：表皮効果による抵抗損失（周波数の平方根に比例）
- $\alpha_d$：誘電損失による減衰（周波数に比例）

#### **高周波での支配的な減衰**
```
減衰量
  ↑
  │         αd（誘電損失）
  │        ╱╱╱╱╱
  │      ╱╱╱╱╱
  │────╱╱╱╱╱
  │   │αc│  ←表皮効果
  └───┴──────────→ 周波数
     0  GHz
     
GHz帯では、αdが支配的
```

#### **実測値と算出値の一致**
- 全基板で実測値と算出値がよく一致
- 北原研究の結論：FR-4基板間の複素誘電率差は伝送特性にほぼ影響しない

## 3. 産総研：極低温下での材料評価技術

### 3.1 研究背景と重要性

#### **量子コンピューターの課題**
- 超伝導回路方式では、極低温の量子ビット制御に高周波信号が必須
- 4K～300K（-269℃～27℃）の広い温度範囲で正確な材料特性評価が必要
- しかし、低温環境での高周波材料特性測定技術が存在していなかった

#### **低温高周波回路の設計課題**
- 高密度化、低損失化が不可欠
- 導電率、誘電率、誘電損失の温度依存性を正確に把握する必要

### 3.2 開発した測定技術の概要

#### **BCDR法の改良**
- 従来：室温でのみ測定可能
- 改良点：
  1. 極低温対応の共振器設計
  2. 温度変化による誤差排除（2つの異なる厚さの共振器を使用）
  3. 4K～300Kで高精度測定を実現

#### **測定対象パラメータ**
```
低温下で測定する3つの材料パラメータ

├─ 比誘電率εr'
│  （信号波長・伝搬速度に影響）
├─ 誘電正接tanδ
│  （信号減衰に影響）
└─ 導電率σ
   （導体損失に影響）
```

### 3.3 重要な発見

#### **導電率の温度依存性**
```
導電率σ
   │        低温では周波数依存性が顕著
   │      ╱╱╱╱╱╱╱╱╱
   │    ╱╱╱╱╱╱╱  300K
   │  ╱╱╱╱╱    
   │╱╱╱╱╱ 4K（依存性強い）
   └──────────────→ 周波数
     室温        高周波
```

#### **導体損失の新知見**
従来の予想：
- 低温→導電率↑→導体損失↓

実際の測定結果：
- 低温→導電率↑ **しかし** 表皮深さ↓
- 表皮深さが銅箔表面粗さと同程度に
- **表面粗さが導体損失の主要因**に変化

#### **低周波での特性**
```
比誘電率εr'
   3.2 │         
       │ ─────── 20K以下でほぼ一定
   3.0 │        
       └──────────→ 温度
         4K  20K  300K

誘電正接tanδ
 0.002│
      │  ╲
      │   ╲
 0.000│    ────── 単調減少傾向
      └──────────→ 温度

```

- 低温環境での高周波部品（増幅器、フィルターなど）の性能予測

### 1.2 測定手順

**① TRL校正**
- Thru（直結）
- Reflect（反射）
- Line（伝送線路）

**② SOLT校正（代替案）**
- Short（短絡）
- Open（開放）
- Load（50Ω負荷）
- Thru（直結）

**③ 測定対象の接続と測定**
- 複数枚の銅箔円板を用意（直径を変えて周波数可変）
- Sパラメータ測定
- Q値から誘電損失を算出

### 1.3 大日方研究でのAFR（自動フィクスチャ除去）

#### **AFRの目的**
コネクタ周辺など不要部分の伝送特性を除去して、DUT（測定対象）部分のみの特性を抽出

#### **AFR手順**
```
Step1: AFR用基板（不要特性のみ）の測定
       →不要な特性データを記録

Step2: 測定対象を含む基板の測定
       →全体の特性データを記録

Step3: PLTS内でAFR処理
       全体 - 不要特性 = DUTの純粋な特性
```

## 3. 測定結果の処理

### 3.1 複素誘電率の計算

#### **BCDR法での計算フロー**

```
実測データ
   ↓
[測定周波数 f0, Q値 Qu, Qc]
   ↓
特性方程式と境界条件の解析
   ↓
[εr'（比誘電率）を決定]
   ↓
[tanδ（誘電正接）を計算]
   ↓
εr'' = εr' × tanδ
   ↓
出力：εr', εr'', tanδ
```

#### **フリースペース法での計算フロー**

```
実測データ
   ↓
[測定周波数、反射/透過Sパラメータ]
   ↓
Keysight N1500（材料測定スイート）
NIST Precisionモデル
   ↓
逆問題を数値解析
   ↓
出力：εr', εr'', tanδ
```

### 3.2 不確かさの評価

#### **北原研究**
- 繰り返し測定（各基板5回）
- 平均値と標準偏差を計算
- メーカ公称値との比較

#### **大日方研究**
- 5回の繰り返し測定
- カバレッジファクターk=2で拡張不確かさを評価
- 各周波数で不確かさを算出

**典型的な不確かさ**：
- εr'：±0.5～0.7%
- tanδ：±8～17%（高周波ほど劣化）

# 第五部：研究結果の考察と応用

## 1. 北原研究の結論

### 1.1 FR-4基板の複素誘電率特性

| 周波数帯 | 比誘電率εr' | 誘電正接tanδ | 特徴 |
|---------|------------|-----------|------|
| **低周波** | 4.0～4.3 | 0.02～0.03 | メーカ差顕著 |
| **中周波** | 3.9～4.2 | 0.02前後 | 周波数低下傾向 |
| **高周波** | 3.8～4.1 | 0.02前後 | ほぼ安定 |

### 1.2 伝送特性への影響評価

**結論**：
- 基板間の複素誘電率差は、伝送特性Sdd21に対してほぼ無影響
- 基板間での減衰差：最大2dB（1.2倍程度）
- 実用上、異なるFR-4基板を選択してもSdd21性能に大差なし

### 1.3 設計上の指針

**高周波回路基板設計時**：
1. FR-4で十分対応可能（テフロン・セラミックは不要な場合多い）
2. コスト最適化：標準的なFR-4で基本的には問題ない
3. クリティカルな回路：複素誘電率の実測確認推奨

## 1. 物理量と単位

| 物理量 | 記号 | 単位 | 説明 |
|--------|------|------|------|
| 周波数 | $f$ | GHz | 1秒間の波数 |
| 角周波数 | $\omega$ | rad/s | $\omega = 2\pi f$ |
| 比誘電率 | $\varepsilon_r'$ | 無次元 | 真空に対する相対値 |
| 誘電損失 | $\varepsilon_r''$ | 無次元 | 複素誘電率の虚部 |
| 誘電正接 | $\tan\delta$ | 無次元 | $\tan\delta = \varepsilon_r''/\varepsilon_r'$ |
| Q値 | $Q$ | 無次元 | 共振鋭度（高いほど低損失） |
| 導電率 | $\sigma$ | S/m | 電気伝導のしやすさ |
| 伝搬速度 | $v$ | m/s | 信号が進む速度 |

## 2. 重要な物理式

### 2.1 複素誘電率

$$\dot{\varepsilon} = \varepsilon_r' - j\varepsilon_r''$$

### 2.2 誘電正接

$$\tan\delta = \frac{\varepsilon_r''}{\varepsilon_r'}$$

### 2.3 導電率から誘電損失への変換

$$\varepsilon_r'' = \frac{\sigma}{\omega \varepsilon_0}$$

### 2.4 伝搬速度

$$v = \frac{c}{\sqrt{\varepsilon_r'}}$$

### 2.5 信号減衰（伝送特性）

$$\alpha = \alpha_c + \alpha_d$$

- $\alpha_c$：表皮効果による抵抗損失
- $\alpha_d$：誘電損失による減衰

# 附録：用語解説

## A. 高周波関連用語

### BCDR（Balanced-type Circular Disk Resonator）
平衡円板共振器。複数のTM0m0モードを活用して、広い周波数範囲での誘電率測定が可能な共振器。

### Sパラメータ（Scattering Parameter）
散乱パラメータ。回路網への信号入力と出力の関係を表す。S11（反射）、S21（透過）など。

### VNA（Vector Network Analyzer）
ベクトルネットワークアナライザ。Sパラメータを測定する高周波測定器。

### TRL校正（Through-Reflect-Line）
直結、反射、伝送線路を用いた高周波測定器の校正方法。マイクロストリップ線路など非同軸系に有効。

### Q値（Quality Factor）
共振鋭度を表す指標。高いほど低損失。$Q = \omega_0 L/R$で定義。

### tanδ（Loss Tangent）
誘電正接。誘電損失の程度を示す。小さいほど損失が少ない。

---

## B. 基板材料関連用語

### FR-4
ガラス繊維（Fiber glass）+エポキシ樹脂（Epoxy）の複合材料。燃焼性グレードの表示。

### MEGTRON
パナソニック製の低誘電率・低損失基板。高周波用途向け。

### 誘電分散
周波数が高くなると、双極子が電界変化に追随できなくなり、誘電率が低下する現象。

### 表皮効果（Skin Effect）
高周波信号が導体表面のみを流れる現象。表皮深さ$\delta = 1/\sqrt{\pi \mu_0 \sigma f}$

---

## C. 測定関連用語

### 導電率σ（Conductivity）
金属の電気伝導のしやすさ。銅の場合、約$5.6 \times 10^7$ S/m。

### 表面粗さ（Surface Roughness）
金属表面の凹凸の程度。高周波では伝導損失に大きく影響。

### キャリブレーション
測定器の系統誤差を除去して正確な測定ができるようにすること。

### 不確かさ評価（Uncertainty Evaluation）
測定結果のばらつきや系統誤差を定量化する。カバレッジファクターk=2で95%信頼度を表現。
