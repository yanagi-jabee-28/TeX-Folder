# スライド2：研究の背景
- AI, 5G/6Gの次世代通信技術は高速で大容量データ送受信が必須
- 高周波ほど通信容量を大きくできる → データセンターなどでは100 GHz帯での通信性能が必要
- 高周波になると信号減衰がある ⇐ 誘電損失 $\alpha_d$ + 導体損失 $\alpha_c$
- 従来：研究室内の誘電損失を測定・検討
- 課題：導体損失における表面粗さの影響の検討は不十分
- 基板と導体を接着するための表面粗さが導体損失に影響？

[数式]
- $\alpha_c = \frac{\pi f \mu}{\sigma Z_0 W}$
- $\alpha_d = \frac{\pi f \sqrt{\epsilon_{eff}}}{c} \tan\delta$

---
# スライド3：表皮効果と表面粗さの関係
- 表皮深さ : $\delta = 1 / \sqrt{\pi f \mu \sigma}$
- 3 GHzで約1.1 μm，30 GHzで約0.36 μm

[図の補足説明]
- 導体と誘電体（基板）の境界部分における電流分布の比較
- 低周波（$\delta$ が深い）：電流が凹凸の奥まで一様に流れる
- 高周波（$\delta$ が浅い）：電流が表面の微細な凹凸に沿って局所的に集中して流れる

# スライド4：本研究の目的
**目的**
- 表面粗さによる電気伝導性への影響を明らかにする。

**やったこと**
- 平衡円板共振器法（BCDR法）を用いてMEGTRON6の複素誘電率測定

- 減衰定数の算出

**測定の流れ**
1. 標準銅箔を使用 → 複素誘電率測定
2. 実基板銅箔を使用 → 導電率測定
3. 減衰定数算出

---
# スライド5：BCDR法による測定方法
**BCDR法（平衡円板共振器法）**
- 誘電体で銅板電極を挟み，圧着
- RF信号入力 → 共振信号を測定 → 評価・計算

**手順**
① エッチング処理で銅箔除去した基板 + 標準銅箔
　→ 基板自体の複素誘電率を測定
② 実基板の銅箔を残した状態で測定
　→ ①の誘電率を基準に実効導電率を算出

# スライド8：測定結果① Sパラメータ
- 20回の測定の平均を表示

- $f_0$ は基板の比誘電率に依存

# スライド9：測定結果② 複素誘電率
- SパラメータのQ値から算出
- 左グラフ：比誘電率（$\epsilon_r'$）はほぼ一定
- 右グラフ：誘電損失（$\epsilon_r'\tan\delta$）は周波数とともに増加

[数式]
- $\dot{\epsilon_r} = \epsilon_r' - j\epsilon_r'\tan\delta = \epsilon_r' - j\epsilon_r''$
- $\tan\delta = K \left( \frac{1}{Q_u} - \frac{1}{Q_c} \right)$

---
# スライド10：測定結果③ 実効導電率
- 周波数が高くなるほど実効導電率が低下
- → 表皮深さの減少と表面粗さの影響

# スライド11：減衰定数の算出結果
- 実効導電率使用時に損失が大きい
- 表面粗さを考慮しない従来モデルが高周波での導体損失を過小評価している可能性
- このような式で実験結果を減衰定数に反映

[数式]
- $Z_0 = \frac{87}{\sqrt{\epsilon_r' + 1.41}} \ln\left(\frac{5.98 H}{0.8 W + T}\right)$
- $\epsilon_{re} = \frac{\epsilon_r' + 1}{2} + \frac{\epsilon_r' - 1}{2} \left(1 + \frac{10H}{W}\right)^{-1/2}$
- $\alpha = \alpha_c + \alpha_d$
- $\alpha_c = 0.173 \frac{\sqrt{\pi f \mu_0 \rho}}{Z_0 W} \, [\text{dB/20mm}]$
- $\alpha_d = 0.173 \frac{f \pi}{c} \epsilon_r'' \sqrt{\frac{\epsilon_{re} - 1}{\epsilon_r' - 1}} \, [\text{dB/20mm}]$
